# California Housing Price Predictor

A Machine Learning web app built using Streamlit that predicts California housing prices based on user inputs.

## Live Demo
[Open Streamlit App](https://california-housing-price-predictor-nzrf2kcbxbdbaekyyuk959.streamlit.app/)

## Tech Stack
- Python
- Scikit-learn
- Pandas
- Streamlit

## Features
- Real-time house price prediction
- Interactive UI
- ML model deployment using Streamlit Cloud
